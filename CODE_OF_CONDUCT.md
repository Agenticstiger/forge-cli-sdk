# Code of conduct

This project adopts the **Contributor Covenant, version 2.1** in full. The canonical text is here:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/

It applies to all project spaces — issues, pull requests, discussions, and any direct project communication — and to behavior in public spaces when an individual is representing the project.

## Reporting

If you experience or witness behavior that violates the code, report it through GitHub:

- For routine reports, **open an issue** at https://github.com/Agenticstiger/forge-cli-sdk/issues with the label `code-of-conduct`. Maintainers will respond.
- For sensitive reports you'd prefer to keep private, use GitHub's **private vulnerability reporting** form at https://github.com/Agenticstiger/forge-cli-sdk/security/advisories/new — it's intended for security but works equally well as a private channel for any sensitive matter.

If your report concerns the maintainer themselves, please escalate to GitHub directly (e.g. GitHub Trust & Safety) — independent escalation is encouraged.

## Enforcement

The Contributor Covenant's *Enforcement Guidelines* (linked from the canonical text above) describe the four-step response ladder maintainers use, from private correction up to permanent ban. The maintainer follows that ladder.
